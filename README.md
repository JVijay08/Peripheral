Peripheral

Peripheral is an interactive web experience that tests how users respond to hidden things on the. 
Each page tracks a different behavior, like mouse movement, attention to screen edges, and 
interaction with hidden records.

How It Works

1. Keep mouse still to reveal the first hidden layer.
2. Keep mouse on center while avoiding the edges.
3. Open each hidden archive record.
4. Receive a behavioral profile based on your interactions.

Built With

- HTML
- CSS
- JavaScript
- Session Storage

Features

- Interactive hidden elements
- Mouse movement tracking
- Edge detection
- File interaction tracking
- Behavior profiles
